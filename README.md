### หมายเหตุ

* ผมเป็นนักแปลมือใหม่ที่ต้องการพัฒนาทักษะในการแปลบทสนทนาที่มีความหลากหลายในเชิงรูปลักษณะประโยค อาทิ เช่น รูปประโยค (phrases), คำ (vocabulary) หรือ วลี (idioms) เป็นต้น

* ทั้งนี้ ผมตั้งใจที่จะแปลไปเรื่อย ๆ ก่อนจะปรับปรุงคำ, บริบท และรูปประโยคในภายหลัง

  > จะเห็นได้ว่า chapter แรก ๆ ตั้งแต่ช่วง Prologue + Canto 1-2 จะยังมีคำที่ไม่เหมาะ หรือความไม่เสถียรในรูปประโยคที่มีการแปลออกไป
